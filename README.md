# Análisis de la tuberculosis

## 📝 Descripción
Este proyecto consiste en un análisis de datos realizado en **Python** utilizando **Google Colab**. El objetivo principal fue analizar la enfermedad infecciosa conocida como tuberculosis  

## 🛠️ Tecnologías Utilizadas
* **Python 3**
* **Pandas:** Para la manipulación de datos.
* **Numpy:** Para calculos matemáticos.
* **Google Colab:** Como entorno de desarrollo.
